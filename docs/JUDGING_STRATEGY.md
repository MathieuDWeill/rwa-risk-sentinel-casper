# Judging strategy

## 90-second framing

"RWA protocols need off-chain monitoring but cannot trust black-box data feeds. We built an autonomous agent that monitors RWA signals, computes explainable risk, stores evidence off-chain, and publishes verifiable risk attestations to Casper so DeFi contracts can react."

## What to show first

1. Dashboard with the asset and current score.
2. Terminal running the autonomous agent.
3. Evidence JSON file and hash.
4. Casper Testnet transaction/deploy hash.
5. Contract read showing latest score.

## What judges should remember

- Not a chatbot.
- Agent takes autonomous action.
- RWA/DeFi use case is obvious.
- Casper transaction is real.
- Roadmap is credible: x402 paid APIs, MCP tools, multi-agent validation.
