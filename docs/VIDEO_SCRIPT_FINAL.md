# Demo Video Script — RWA Risk Sentinel

Target length: 2:30 to 3:00.

## Hook

AI agents are starting to make decisions about real-world assets. But in DeFi, an AI risk score is useless if nobody can verify what the agent saw, decided, and published.

## Solution

RWA Risk Sentinel is an autonomous AI agent that reads off-chain RWA signals, computes a deterministic risk assessment, hashes the evidence trail, and publishes a verifiable attestation on Casper Testnet.

## Demo

Select invoice-2026-001.

The agent detects payment delay, weak debtor credit quality, macro volatility, collateral drawdown, and negative news.

The result is a HIGH risk score with a confidence level and an evidence hash.

Because the risk is material, the agent publishes the attestation to Casper Testnet.
