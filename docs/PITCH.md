# Pitch

## Name

RWA Risk Sentinel

## One-liner

An autonomous RWA risk oracle that publishes explainable, evidence-backed attestations to Casper.

## Problem

RWA DeFi needs continuous off-chain monitoring, but off-chain data is fragmented, hard to audit, and often not available to smart contracts in a useful form.

## Solution

An autonomous agent monitors real-world asset signals, computes a risk score, generates an evidence package, hashes it, and writes the compact attestation to Casper Testnet.

## Why now

Agentic AI makes continuous monitoring and decision-making practical. Casper provides a strong foundation for upgradable, enterprise-friendly smart contracts and agent-oriented infrastructure.

## Target users

- RWA lending protocols
- Invoice financing platforms
- Insurance pools
- DAO treasuries holding off-chain assets
- Data providers monetizing risk feeds

## Roadmap

- Qualification: working agent + Testnet attestation.
- Final: real data source + MCP server + x402 paid API + multi-agent validation.
- Post-buildathon: partner with RWA/DeFi protocols and launch as risk infrastructure.
